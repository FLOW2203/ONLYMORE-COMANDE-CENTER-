# onlymore-investor

Skill de génération de documents investisseurs pour ONLYMORE Group.

## Contexte

ONLYMORE Group (holding) lève des fonds pour ses filiales :
- **COLHYBRI** : app de commerce local hybride (€3/mois, TAM France 19.3M)
- **CROWNIUM** : plateforme sport / fan engagement
- **DOJUKU SHINGI** : communautés inter-culturelles (US launch Rust Belt)
- **ONLYMORE FINANCE** : services financiers du groupe

Valorisation pré-seed : €1.3M — €2.15M

## Instructions

Quand ce skill est invoqué :

### 1. Pitch Deck
Générer un pitch deck structuré (14 slides max) :
1. Cover — nom, tagline, logo
2. Problème — solitude urbaine, commerce local en déclin (stats sourcées)
3. Solution — plateforme hybride digitale + physique
4. Produit — screenshots, features clés
5. Traction — métriques réelles (pas de vanity metrics, pre-seed)
6. Business model — abonnement €3/mois, projections 3 ans
7. Marché — TAM/SAM/SOM avec sources
8. Concurrence — matrice de positionnement
9. Équipe — fondateur + agents IA
10. Go-to-market — France d'abord, puis España Vaciada, US Rust Belt
11. Financials — projections revenus, burn rate, runway
12. Ask — montant recherché, utilisation des fonds
13. Roadmap — milestones 12 mois
14. Contact — coordonnées, liens

### 2. Dossier Financier
- Projections revenus sur 3 ans (conservateur / base / optimiste)
- Structure de coûts (infra, marketing, RH)
- Unit economics (CAC, LTV, churn)
- Apports en nature valorisés (IP, code, design)

### 3. Outreach Investisseurs
- Template email personnalisable (FR + EN)
- Message LinkedIn court (< 300 caractères)
- Tweet/post X pour visibilité
- Adapter le ton selon le type de VC (early-stage, impact, sport-tech)

### 4. Formats de sortie
- Markdown structuré (pour conversion PPTX/PDF)
- Textes prêts à copier-coller
- Chiffres toujours sourcés ou marqués comme projections
