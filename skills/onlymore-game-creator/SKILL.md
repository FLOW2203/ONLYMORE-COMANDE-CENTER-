# onlymore-game-creator

Skill de création de mini-jeux HTML5 pour l'engagement communautaire ONLYMORE Group.

## Contexte

Les filiales ONLYMORE utilisent des mini-jeux interactifs pour :
- Engager les utilisateurs sur les plateformes (COLHYBRI, CROWNIUM)
- Gamifier les interactions communautaires
- Créer du contenu viral pour les réseaux sociaux

## Instructions

Quand ce skill est invoqué :

### 1. Choix du type de jeu
Proposer un des templates suivants selon la filiale :
- **Quiz culturel** (COLHYBRI) : questions sur la culture locale, traditions, gastronomie
- **Défi sportif** (CROWNIUM) : quiz sport, pronostics, trivia compétitions
- **Memory cards** : associer des éléments visuels (logos filiales, drapeaux, produits)
- **Wheel of fortune** : roue interactive avec récompenses / codes promo
- **Puzzle slider** : image à reconstituer (branding filiale)

### 2. Génération du code
- Créer un fichier HTML unique (standalone, aucune dépendance externe)
- Utiliser Canvas API ou DOM pur pour les animations
- Design responsive (mobile-first)
- Palette de couleurs ONLYMORE :
  - COLHYBRI : `#00D4AA`
  - CROWNIUM : `#FFD700`
  - DOJUKU SHINGI : `#FF6B35`
  - GROUPE : `#7B61FF`
- Score tracking avec localStorage
- Bouton de partage social (copier le score)

### 3. Intégration
- Le jeu doit pouvoir être intégré via `<iframe>` sur n'importe quel site
- Fournir le code embed ready
- Taille recommandée : 400x600px (portrait mobile)

### 4. Livrables
- `index.html` — le jeu complet
- Instructions d'intégration
- Capture d'écran ou description du gameplay
