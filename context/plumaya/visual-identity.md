## Typographie

### Hierarchie
| Niveau | Police | Style | Usage |
|--------|--------|-------|-------|
| Titres editoriaux | [A COMPLETER] | Serif elegante, lisible, "publishing feel" | Couvertures, titres de chapitres |
| Sous-titres | [A COMPLETER] | Serif ou sans-serif semi-bold | Sections, sous-chapitres |
| Corps de texte (print) | [A COMPLETER] | Serif classique | Livres, manga, BD |
| Corps de texte (digital) | [A COMPLETER] | Sans-serif lisible | Site web, reseaux sociaux |
| Accents decoratifs | Typographie maya stylisee | Display/ornementale | En-tetes PLUMAYA, logos, decorations |

### Regles Typographiques
- Interlignage minimum : 1.5x pour le digital, 1.3x pour le print
- Ne jamais utiliser la typo maya stylisee pour du texte courant
- Les titres de livres utilisent toujours la serif editoriale, jamais du sans-serif

## Style Illustratif

### Signature PLUMAYA
- Melange tradition / technologie : illustrations qui montrent la cohabitation du fait main et du digital
- **Style maison** : aquarelle digitale, textures papier, couleurs riches
- Chaque univers IP a son propre style (voir guides respectifs)

### Par Univers
| Univers | Style illustratif | References |
|---------|-------------------|------------|
| DOJUKU SHINGI | Manga japonais traditionnel, lignes dynamiques, encre noire + couleurs vives | Naruto, Vagabond, Baki |
| COLHYBRI | Illustration jeunesse, couleurs chaudes, traits arrondis, personnages expressifs | Miyazaki, albums Gallimard Jeunesse |
| CROWNIUM | Photo-realiste augmente, docu-style, textures terrain/stade | Docu-series Netflix, The Athletic |

### Pipeline Illustration
1. **Brief visuel** : description de la scene, ambiance, references
2. **Croquis humain** (optionnel) : esquisse de base pour guider l'IA
3. **Generation IA** : ComfyUI / Gemini / Nano Banana selon l'univers
4. **Retouche humaine** : correction des artefacts, ajustement des details
5. **Validation editoriale** : verification de coherence avec l'univers IP
6. **Export** : formats adaptes (print haute resolution, web optimise)

### Regles de Mise en Page
- Marges generales : 15-20mm pour le print, 20-40px pour le digital
- Grille : 12 colonnes pour le web, libre pour le print
- Les illustrations ne sont jamais recadrees au point de perdre le sujet principal
- Toujours laisser de l'espace negatif autour du texte (respiration editoriale)

## Interdits Visuels

1. Pas de "look IA" evident (artefacts, mains deformees, yeux morts)
2. Pas de style corporate/froid
3. Pas d'uniformisation : chaque IP garde son univers visuel propre
4. Toujours une touche humaine visible (texture, imperfection volontaire)
5. Jamais de photos stock generiques
6. Pas de clipart ou illustrations de basse qualite
7. Ne jamais mixer les palettes de deux univers IP sur un meme visuel
