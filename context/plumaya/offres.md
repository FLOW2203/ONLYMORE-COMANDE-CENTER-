# PLUMAYA Editions -- Offres & Modele Economique

## Vue d'Ensemble

PLUMAYA est la maison d'edition IA-native du groupe ONLYMORE. Elle exploite les univers narratifs (IP) des filiales du groupe pour creer des produits editoriaux multi-formats : livres, manga, animation, produits derives.

## Verticales d'Edition

### 1. Livres (Print + Digital)

| Element | Detail |
|---------|--------|
| Format | Livres illustres, romans, guides |
| Distribution | Amazon KDP (impression a la demande), librairies independantes [A COMPLETER] |
| Prix type | 12-25EUR selon format |
| Marge KDP | ~35-70% selon canal (KDP direct vs expanded distribution) |
| Outils | KDP Amazon, InDesign, Canva (brand kits ONLYMORE) |

**Pipeline de production** :
1. Ecriture : Claude (redaction assistee) + auteur humain
2. Revision : relecture humaine + verification IA
3. Illustration : ComfyUI / Canva selon le style
4. Mise en page : InDesign / Canva
5. Publication : upload KDP, validation, lancement

### 2. BD / Manga

| Element | Detail |
|---------|--------|
| Format | Series manga (DOJUKU SHINGI), BD jeunesse (COLHYBRI) |
| Distribution | Amazon KDP, plateformes manga digitales [A COMPLETER] |
| Pipeline illustration | ComfyUI, Gemini, Nano Banana |
| Tome 1 DOJUKU SHINGI | [A COMPLETER statut] |
| Prix type | 9-15EUR par tome |

**Specificites manga DOJUKU SHINGI** :
- 5 tomes prevus (1 par maitre/culture martiale)
- 81 techniques illustrees a integrer dans la narration
- Style visuel : manga japonais traditionnel augmente IA

### 3. Animation (YouTube / Plateformes)

| Element | Detail |
|---------|--------|
| Format | Series courtes animees (3-10 min), trailers, clips promotionnels |
| Outils | Unity (animation), ElevenLabs (voix), Kling 2.5 (video IA) |
| Distribution | YouTube (monetisation), plateformes streaming [A COMPLETER] |
| Premiere serie cible | DOJUKU SHINGI animated shorts |
| Monetisation | AdSense YouTube, sponsoring, integration catalogue |

**Pipeline animation** :
1. Storyboard : script + decoupage scene par scene
2. Assets : generation des personnages et decors (ComfyUI/Gemini)
3. Animation : assemblage Unity / montage video
4. Voix : narration et dialogues via ElevenLabs
5. Post-production : sound design, titrage, export
6. Publication : upload YouTube, optimisation SEO video
